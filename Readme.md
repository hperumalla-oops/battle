## RPi programs (Always work in a virtual environment.)

###  pip
`sudo apt install python3-pip`

###  venv
`sudo apt install python3-venv`

`python -m venv myenv`

`source myenv/bin/activate`

###  Adafruit_DHT   
`python -m venv myenv`

`source myenv/bin/activate`

`pip3 install adafruit-circuitpython-dht`
